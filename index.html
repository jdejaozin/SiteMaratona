{ Criatividade, trabalho em equipe e descoberta de talentos! }

Objetivos

A Primeira Maratona De Programação Interdisciplinar tem como objetivo promover nos alunos a criatividade, capacidade de trabalho em equipe, habilidade de resolver problemas sob pressão e possibilitar a descoberta de novos talentos.
O público alvo são os alunos dos cursos de ADS e EC. A maratona será ranqueada por níveis, de acordo com o conhecimento dos alunos e acontecerá no dia 11 de Julho(sábado) das 13h30 às 17h.

Níveis da Maratona

Noob


Requisitos:
cursar matéria de Algoritmos

Conhecimento necessário:
C / C++

A maratona possui 5 diferentes níveis para que todos os alunos possam participar sem ocorrer disparidade de níveis.
ATENÇÃO: para participar é necessário estar cadastrado no site Beecrowd.

Conheça os outros níveis abaixo

Pilha


Requisitos:
cursar Estrutura de Dados



Conhecimento necessário:
C / C++

Pilha na Web


Requisitos:
cursar Linguagem de Script Web (ADS) ou Tecnologias para Aplicações Web (ADS)

Conhecimento necessários:
JavaScript

Ordenado


Requisitos:
cursar Pesquisa e Ordenação (ADS, EC) ou Programação Orientada a Objetos (ADS, EC)

Conhecimento necessário:
C / C++ / Java / JavaScript

Profissional


Requisitos:
qualquer estudante de ADS ou EC


Conhecimento necessário:
C / C++ / Java / JavaScript

Certificados

Serão emitidos certificados após a conclusão da Maratona, para que possam ser contabilizadas em horas complementares e servirem como comprovante para experiência em eventos competitivos.
